The utility used to test the functionality of scales Dymo M5 and M10, or for simple weighing.
Allows the weight value is automatically saved to the clipboard, and eg. in Excel simply paste CTRL + V.
Copy format can be configured either as a number or numbers and units.

The program was written in Delphi, 2010.
For compilation requires JEDI Components Delphi http://www.delphi-jedi.org/
Or just a single component JvHidDeviceController
